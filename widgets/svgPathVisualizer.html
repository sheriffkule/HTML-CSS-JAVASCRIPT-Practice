SVG Path Visualizer Help

SVG Path Commands

  • M/m (MoveTo): Start a new sub-path at (x, y).
  • L/l (LineTo): Draw a line to (x, y).
  • H/h (Horizontal LneTo): Draw a horizontal line to x.
  • V/v (Vertical LineTo): Draw a vertical line to y.
  • C/c (Cubic Bezier): Draw a cubic Bezier curve to (x, y).
  • S/s (Smooth Cubic Bezier): Draw a smooth cubic Bezier to (x, y).
  • Q/q (Quadratic Bezier): Draw a quadratic Bezier to (x, y).
  • T/t (Smooth Quadratic Bezier): Draw a smooth quadratic Bezier to (x, y).
  • A/a (Elliptical Arc): Draw an elliptical arc to (x, y).
  • Z/z (ClosePath): Close the current sub-path.

    • Uppercase commands use absolute coordinates, lowercase uses relative coordinates.
